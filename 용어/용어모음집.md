* AKS : Azure Kubernetes Service, Azure에서 제공하는 Managed Kubernetes(Container Ochestration) Service, Docker 컨테이너를 자동으로 배치 / 확장 / 업데이트 수행해주는 Azure의 Kubernetes 클러스터
* Kubernetes : Container를 Ochestration 수행하기 위한 플랫폼. 자원 사용에 따라 Auto Scaling 수행함. 결국, Docker는 컨테이너를 포장하기 위한 도구인 것이고, k8s(Kubernetes)는 포장된 컨테이너들을 자동 관리하기 위한 플랫폼
* Azure Functions : Serverless 함수 실행 서비스. 특정 이벤트가 발생했을 때 자동으로 실행되는 코드 조각.
* Trigger : 무엇이 Azure Functions을 실행 시키는가?
* * HTTP Trigger : HTTP 요청 수신시 실행 -> API 기능 만들기
* * Timer Trigger : Scheduling (CRON)
* * Blob Trigger : Blob Storage에 파일 생성 혹은 변경
* * Queue Trigger : Queue Storage 메시지 -> 비동기 작업 처리
